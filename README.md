# SADU TYPING | Neural Interface
> **SADU Typing** is an elite, minimalistic typing training application designed for precision and speed. Developed under the **YGYT** technology framework.

---

## 🛠️ System Overview
SADU Typing is built to simulate a high-tech neural link. Users progress through **50 specific phases** of increasing difficulty, moving from basic muscle memory to complex technical vocabulary.

### Key Features:
* **50 Progressive Levels:** Structured curriculum to build typing mastery.
* **Real-time Analytics:** Instant tracking of **Precision** (Accuracy) and **Velocity** (WPM).
* **Neural Interface UI:** A professional, dark-mode design inspired by Netflix and Google aesthetics.
* **Verification Protocol:** Automated email certification system triggered upon completion of the final phase.
* **Anti-Spam Security:** One-time claim lock for official certificates.

---

## 🚀 Tech Stack
* **Backend:** Python 3 with Flask
* **Database:** SQLAlchemy (SQLite)
* **Email Engine:** Flask-Mail (SMTP)
* **Frontend:** Vanilla JavaScript (ES6+), CSS3 with Glitch effects, HTML5 Canvas
* **Security:** Python-Dotenv for environment variable protection
